Equilibrium Game Engine

Equilibrium is an original Physics based Game Engine,allowing users to freely create simple or more complicated game levels. It proposes the base input, phusics and graphic systems but is also highly customizale. The engine itself is written on C++ and using OpenGL libraries.
Full demonstration video : A simple level created on Equilibrium Game Engine

Catchy Punchy Punch

CPP is a rhythmic game dictated by the arrival of enemies. Its 2.5D arcade style imposes the main goal of the game: survive as long as possible!
The game is written on Unity and the source code is on GitHub.

HabitiMia

Gamify your life!
HabitiMia is a mobile application for habits and productivity, allowing the user to set up tasks and complete them just like quests in a game, but in real life.
Full demonstration video : HabitiMia

Facial Recognition System

This project proposes a facial recognition system with three different complexities :
  • ligthweigth real-time recognition with lesser quality of recognition
  • fast and accurate real-time recognition
  • highly fairly accurate image based recognition
