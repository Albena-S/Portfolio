About Me

I am very passionate about Virtual and Augmented Reality technologies and Image processing and analysis, both 3D and 2D. Being able to create an experience that cannot be felt in any other way is very motivating and even thrilling to me.
This field of work is expanding rapidly and I want to be a part of that growth!

I am currently in my last year of the Master's degree IMAGINE (Images, Games and Intelligent Agents) at the University of Montpellier. My program aims to train engineers and researchers specialized in image and vision industries and laboratories, computer graphics, virtual and augmented reality, video games and interactive simulation.

Projects

Here are some of the projects I've worked on in the last year - they vary in theme, programming language, and logic.

Equilibrium Game Engine

Equilibrium is an original Physics based Game Engine,allowing users to freely create simple or more complicated game levels. It proposes the base input, phusics and graphic systems but is also highly customizale. The engine itself is written on C++ and using OpenGL libraries.
Full demonstration video : A simple level created on Equilibrium Game Engine

Catchy Punchy Punch

CPP is a rhythmic game dictated by the arrival of enemies. Its 2.5D arcade style imposes the main goal of the game: survive as long as possible!
You can play on Itch.io!
The game is written on Unity and the source code is on GitHub.

Tetrus against an AI

It is a simple block placement game, inspired by Tetris, with a twist!
The player's main goal is to prevent a squirrel from reaching a nut by placing ice blocks in the most inefficient way, as judged by traditional Tetris rules. The squirrel, an AI agent, on the other hand, is hungry and instantly adapts to find the best block placement to destroy the lines of ice cubes to eat the nut at the bottom of the ice.
Project presentation on Canva.

OpenGL mini-projects

The projects I did using GLSL covered some of the following topics : Shadow maps, Ray tracing, Parametric surfaces, Volume projection, 3D image manipulation and many more.

Facial Recognition System

Metal Gear Solid VI : Return of the Cat

This is a multi-level hide & escape type of game. Each level has a different environement and varied number of AI agents, which all use the principle of the Blackboard system.
Source code on GitHub.

Image Processing and Analysis

I did multiple mini-projects covering the basics of image analysis, digital image processing and image encoding and compression.
Some of the topics studied for image processing are Enhancement, Restoration, Correction, Smart Blur, Edges detection, Morphological operations such as thresholding, closing and opening of a binary image, extension to grayscale images.
As for image encoding and compression I went over coding by contours, regions or shape, Predictive coding and Color space change - RGB, YCrCb or Lab.
